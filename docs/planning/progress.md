## 核心决策记录

### 技术栈选型

| 层级 | 决策 | 理由 |
|------|------|------|
| 小程序 | uni-app + Vue 3 | 一套代码输出微信小程序，国内生态好 |
| PC 管理端 | Vue 3 + Element Plus | 与小程序同技术栈，组件库成熟 |
| 后端 | Java Spring Boot 3 + MyBatis-Plus | 团队 10 年 Java 经验，WxJava SDK 成熟 |
| 数据库 | MySQL 8.0 | 团队最熟悉，JSON 字段满足需求 |
| 迁移 | Flyway | Java 生态标准方案 |
| 部署 | Docker Compose | 单机 2C4G，简单运维 |

### 业务设计决策

| 决策 | 说明 |
|------|------|
| 小程序登录 | 微信一键获取手机号（`wx.getUserPhoneNumber`） |
| PC 登录 | 固定密码登录 + 首次强制修改 |
| 支付 | v1.0 预留接口，v1.5 接入小程序支付 |
| 管理端 | 同一小程序通过 `role` 字段区分用户端/管理端 |
| 架构模式 | 领域分层（domain/application/infrastructure/controller）|
| 主键 | BIGINT 自增（非 UUID） |
| 图片 | `images` 数组第一个元素为封面图 |
| 订单幂等 | 前端防抖 + 后端时间段冲突检测 |
