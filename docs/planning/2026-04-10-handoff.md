## 今日完成

### 1. 后端编译修复
- `AuthService` 移除 `BCryptPasswordEncoder`（改为 SHA-256 简化编码）
- `Order.java` 添加 `@Setter`（缺少 setter 导致大量编译错误）
- `WxJava API` 修正：`getUserPhoneNumber` → `getNewPhoneNoInfo`
- 控制器泛型修复：`ApiResponse<Map<String, String>>` → `ApiResponse<?>`
- ✅ 后端 45 个 Java 文件全部编译通过

### 2. 产品需求更新
- 用户故事从 14 个扩展到 25 个（新增 US-15 ~ US-25）
- 完成 4 阶段 Sprint 规划
- 需求池文档（backlog.md）按 Sprint 重新组织

### 3. 架构设计更新
- 图片存储方案：base64 存数据库（MVP）→ v1.5 迁移 OSS
- 定价方案：MVP 简化版（`days × weekday_price`）→ v1.5 完整 PricingEngine
- 订单-车辆关联：两查询 + 应用层组合
- 前端页面架构：11 个页面路由映射
- 新增 ADR-008/009/010（后端算价、简化定价、硬编码取车地址）

### 4. API 接口文档更新
- 完整覆盖 10 个模块（Auth、Vehicle、Pricing、Order、Agreement、Admin 等）
- 每个接口包含参数、响应、错误码
- 明确标注 MVP vs 延期
