# 需求池 (Backlog)

## Sprint 1: Demo（小程序用户端核心流程）

### US-01: 模拟登录（Demo 阶段）

**用户故事**: 作为租客，我希望在开发环境中能快速登录小程序，以便测试功能。

**验收标准**:
- [ ] 登录页保留微信授权按钮（UI 占位）
- [ ] Demo 模式：点击按钮直接获取模拟 token（bypass 微信校验）
- [ ] 登录成功后跳转到首页
- [ ] 真实微信登录接口预留（Sprint 3 接入）

**优先级**: P0 | **状态**: 待开发 | **预估**: 1h
